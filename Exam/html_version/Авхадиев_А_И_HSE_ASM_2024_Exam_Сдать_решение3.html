Задача 3-Decompile: Декомпиляция программы с массивом (60 баллов)

Даны два фрагмента кода на языке Си c пятью пропусками:

// объявление глобальных переменных пропуск_1 a; short A[23][9][пропуск_2]; int x, y, z; // выражение-оператор пропуск_3 = A[пропуск_4][пропуск_5][y];

Для этих фрагментов компилятор построил следующий код:

mov eax, dword[z] shl eax, 3 add eax, dword[z] add eax, dword[x] mov edx, dword[y] lea eax, [eax + eax * 2] lea eax, [edx + eax * 2] movsx eax, word[A + eax * 2] mov edx, dword[a] mov dword[edx], eax

Вам необходимо восстановить пропуски в исходном фрагменте кода на языке Си. Выпишите их в ответе по одному на строке. В ответе должно быть ровно пять строк. Если не получается заполнить какой-то из пропусков, оставьте эту строку пустой.

На месте пропуска может быть либо число (в таком случае выписывайте число в десятичной системе счисления), либо одно из выражений, указанных ниже под буквами (в таком случае выписывайте соответствующую букву):

  1. short
  2. int
  3. unsigned short
  4. unsigned int
  5. short *
  6. int *
  7. unsigned short *
  8. unsigned int *
  9. a
  10. A
  11. x
  12. y
  13. z
  14. *a
  15. *A
  16. *x
  17. *y
  18. *z
  19. x + y
  20. x + z
  21. y + z
  22. x * 2
  23. y * 4
  24. z * 9

Пример форматирования ответа:

100 A 200 B 50
